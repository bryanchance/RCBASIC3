CalculateFaceZ(cam_dist, graph_offset_x, graph_offset_y, view_w, view_h, view_depth, mA, f_vertex_count, ByRef columns, ByRef face_min_z, ByRef face_max_z, ByRef z_avg)

Calculates a polygons position from a camera and returns the min and max z values as well as the average.

Parameters:
  cam_dist - Camera distance from clipping plane
  graph_offset_x - Where X is 0 ( generally the center of the view )
  graph_offset_y - Where Y is 0 ( generally the bottom of the view )
  view_w - The width of the view   view_h - The height of the view   view_depth - The max z depth of the view   mA - A matrix containing the vertices for the face
  f_vertex_count - the number of vertices ( NOTE: it must be 3 or 4 )
  columns - An array specifying the columns in the matrix for each vertex
  face_min_z - The minimum z value for the face
  face_max_z - The maximum z value for the face
  z_avg - The average of the z values
