WindowFlags ( win )

Returns the flags set on the given window as a bitmask. The mode set with WindowOpen() is part of this mask.

Example:
   my_window_mode = WindowMode(1,0,1,0,0)
   WindowOpen ( 0, "My App", WINDOWPOS_CENTERED, WINDOWPOS_CENTERED, 640, 480, my_window_mode, 1 )

   Print"The flags on the window are ";WindowFlags(0)
